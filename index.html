SERVER FEATURE

Land Protection

Bangun kerajaanmu tanpa rasa takut. Sistem proteksi lahan di server ini menggunakan sistem yang mudah dipahami. Amankan rumah, chest, dan ternakmu dari griefer secara otomatis dan permanen. Ketenangan pikiran 100%.

Rank

Dapatkan identitas eksklusif dengan prefix warna-warni yang menonjol di dalam chat global. Nikmati akses prioritas di server, dan mendukung revine smp dengan cara membeli rank. Buka berbagai kit spesial serta command unik yang mempermudah petualanganmu setiap hari.

Multi-Sethome

Jangan takut tersesat. Tandai berbagai titik penting di dunia dan lakukan teleportasi kilat kapan saja.

TPA System

Mabar makin gampang. Kirim request teleportasi ke temanmu secara langsung dengan aman dan cepat.

Sit & Lay Down

Duduk di atas tangga, slab, atau rebahan di rumput mana saja tanpa mod tambahan.

Komunitas Solid & Bebas Toxic

Kami memiliki sistem anti-cheat ketat dan staff aktif yang siap menjaga lingkungan bermain tetap positif, aman, dan menyenangkan bagi semua kalangan umur.

TUTORIAL

Cara Bergabung

01

Buka Multiplayer

Jalankan Minecraft Java (Versi latest), klik menu Multiplayer, lalu pilih Add Server.

02

Masukkan IP

Isi Server Address dengan NOT YET REALESE. Klik Done.

03

Register & Login

Masuk ke server, amankan akunmu dengan mengetik /register [password] [password] di chat.

01

Menu Server

Buka Minecraft Bedrock (HP/Console), masuk ke tab Servers, scroll ke bawah, klik Add Server.

02

Isi Data Server

Server Name: Revine.
Server Address: NOT YET REALESE.
Port: NOT YET REALESE.

03

Login Akun

Sama seperti Java, amankan akunmu dengan perintah /register saat pertama kali masuk ke dalam lobi.
